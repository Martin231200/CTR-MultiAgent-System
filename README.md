# 🤖 Predictor CTR Multi-Agente (LangGraph)

Este sistema utiliza una arquitectura de agentes autónomos para procesar datos y predecir el Click-Through Rate (CTR).

## 🛠️ Tecnologías utilizadas
* **LangGraph**: Orquestación de agentes.
* **Scikit-Learn**: Modelado estadístico y comparativa de ML.
* **Gradio**: Interfaz de usuario para despliegue web.

## 📊 Resultados obtenidos
El sistema comparó modelos automáticamente y seleccionó la **Regresión Lineal** con un **MSE de 0.0180**.
